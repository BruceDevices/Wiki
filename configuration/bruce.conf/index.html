    bruce.conf#

    This file is used to persist settings between boots and when swapping firmware on your device when using Launcher when you finish running other firmware.

    TODO: Don't store pwd in cleartext :P

    If there is no bruce.conf file on the LittleFS or SD card (if available) then one will be created.
