# Cardputer - CC1101/NRF24

## Items Needed

* [CC1101](../../external-modules/cc1101.md#modules)
* [NRF24](../../external-modules/nrf24.md#modules)
* [Micro SD Sniffer](../../external-modules/micro-sd-sniffer.md#modules)


### Pins

| CC1101/NRF | Cardputer / SD Sniffer |
| -: | - |
| GND - 1 | GND (on SD Sniffer) |
| VCC - 2 | VCC (on SD Sniffer) |
| GDO0 - 3 | G2 (on Cardputer) |
| CSN - 4 | G1 (on Cardputer) |
| SCK - 5 | CLK (on SD Sniffer) |
| MOSI - 6 | CMD (on SD Sniffer) |
| MISO/GDO1 - 7 | DAT0 (on SD Sniffer) |
| DGO2 - 8 | Not Connected |
