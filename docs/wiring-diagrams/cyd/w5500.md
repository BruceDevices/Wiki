## W5500

### Pins

| W5500 | CYD |
| -: | - |
| GND - 1 | GND (on SD Sniffer) |
| SCLK - 2 | CLK (on SD Sniffer) |
| INT - 3 | 22 |
| SCS - 4 | 27 |
| RST - 5 | Not Connected |
| MOSI - 6 | CMD (on SD Sniffer) |
| GND - 7 | Not Connected |
| MISO - 8 | DAT0 (on SD Sniffer) |
| 5V - 9 | Not Connected |
| 3V3 - 10 | VCC (on SD Sniffer) |
