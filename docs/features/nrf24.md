# NRF24

![NRF24 Main Menu](nrf24-main-menu.png)

Use an NRF24 (or GT24) module to jam 2.4GHz signals and view the spectrum.

![NRF24](nrf24.png)


## Module

You will need a [NRF24](../external-modules/nrf24.md) module to use these features, some devices do include a built in NRF24.
